#Xcode snippets

##Instructions

1. git clone https://github.com/MoralAlberto/xcode-snippets.git xcode-snippets
2. cd xcode-snippets
3. Run the install command: ./install
4. Restart Xcode

##Snippets

####Literal String

Shortcut | Snippet
:------------: | :------------:
ls | @"<#something#>"

####Literal Array

Shortcut | Snippet
:------------: | :------------:
la | @[<#something#>]


####Literal Dictionary

Shortcut | Snippet
:------------: | :------------:
ld | @{<#key#>:<#value#>}

####Key value

Shortcut | Snippet
:------------: | :------------:
kv | <#key#>:<#value#>
